## Descripción del proyecto
En este repositorio se presenta una aplicación desarrollada en C# que permite realizar vuelos en entornos interiores. En ella se permite diseñar manualmente el espacio, volar manualmente o diseñar y ejecutar planes de vuelo. También se permite capturar imágenes y vídeos durante la ejecución y controlar el dron mediante un joystick externo.

El proyecto principal consta de todos los formularios que componen la interfaz gráfica, como la estación de tierra, el editor de planes de vuelo o el editor de espacios interiores. Estos formularios interactúan con una biblioteca de clases, que contiene todas las funciones que permiten el control del dron, y que está basada en la biblioteca csDronLink.

## Instrucciones de instalación
A continuación se detallan los pasos a seguir para poder descargar el repositorio de manera local:
1. Descargar la aplicación **Git**. Puedes decargarla desde [git-scm](https://git-scm.com/install/windows).
2. Una vez descargada, abrir un **Git Bash** en la carpeta donde se quiere clonar el repositorio.
3. Clonar el repositorio desde la consola mediante el siguiente comando:
   
   ```bash
   git clone https://github.com/MiguelMartinBenitez/IndoorFlightPlanner

### Requisitos previos
Algunos de los requisitos previos necesarios para el correcto uso de la aplicación son los siguientes:

- Tener instalada la aplicación de escritorio **Microsoft Visual Studio**. Puedes descargarla desde [Visual Studio Code](https://visualstudio.microsoft.com/es/downloads/).
- Tener instalado el simulador de vuelos **Mission Planner**. Aunque su instalación no es estrictamente necesaria, se recomienda para realizar vuelos simulados. Puedes descargarlo desde [Mission Planner](https://ardupilot.org/planner/docs/mission-planner-installation.html).
